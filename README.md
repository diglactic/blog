# Diglactic Newsroom

## Install on Virtual Box
```shell script
apt-get install ruby-full
git clone https://github.com/diglactic/newsroom.git newsroom
cd newsroom
bundle install
jekyll serve -H 192.168.1.100 --force_polling
```
