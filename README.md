# Diglactic Blog

## Install on Virtual Box
```shell script
apt-get install ruby-full
gem install bundler
git clone https://github.com/diglactic/blog.git blog
cd blog
bundle install
jekyll serve -H 192.168.10.10 --force_polling
```
